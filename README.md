React boilerplate for personal purposes.

#### Features

- React 16
- Webpack 4
- Babel 7
- Hot Module Replacement
- Dev + Production Builds
- Source Maps
- ESLint + Prettier

#### Scripts

##### Run the project in development mode

```
npm run start
```

##### Compiles and minify for production

```
npm run build
```

##### Lint your code

```
npm run lint
```
